# Plant UML

Play around online: [www.plantuml](www.plantuml)

## Class Diagram

By default an element is a class.

Other elements are:

- Interface: `interface MyInterface`
- Abstract: `MyAbstractClass`
- Entity: `entity MyEntity`

Relationships between classes:

- Inheritance: `--|>`
- Composition: `--*`
- Aggregation: `--o`
