Build Linux Kernel (BeagleBoneBlack - BBB)

Install toolchain

  • Compiler gcc-arm-linux-gnueabi 

    sudo apt-get install gcc-arm-linux-gnueabi

  • GIT 

    sudo apt-get install git
git config --global user.name "hamajo75"

  • lzop Compression Tool 

    sudo apt-get install lzop

  • mkimage of U-Boot bootloader (image format uImage) 

    sudo apt-get install libssl-dev

    Download U-Boot, make and install U-Boot tools: 

    wget ftp://ftp.denx.de/pub/u-boot/u-boot-latest.tar.bz2 
tar -xjf u-boot-latest.tar.bz2
cd into u-boot directory
make sandbox_defconfig tools-only
sudo install tools/mkimage /usr/local/bin

Compile the Kernel

  • Get the source 

    git clone git://github.com/beagleboard/linux.git
cd linux 
git checkout 4.1

  • Build it

    Notes:

    • CROSS_COMPILE specifies the ARM-toolchain
    • bb.org_defconfig config file for building the kernel
    • -j4 tells make how many jobs to run in parallel (~ nr. of cpu cores)
    • uImage is the type of image U-Boot requires Notes:
      Image
      generic kernel binary image file
      zImage
      compressed, self-extracting Image
      uImage
      image file with U-Boot wrapper (64-bytes), installed by mkimage
    • dtbs build the device tree according to the rules in arm/boot/dts

    Set up configuration: 

    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- bb.org_defconfig

    Build the kernel: 

    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- -j4

    Build device tree, make image: 

    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- uImage dtbs LOADADDR=0x80008000 -j4

    Build kernel modules 

    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- modules -j4

  • Install the modules to the rootfs 

    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- INSTALL_MOD_PATH=/home/cpeacock/export/rootfs modules_install

Raspberry Pi

Temperature Sensor DS1820

Load required drivers for 1-wire interface: 

sudo modprobe wire
sudo modprobe w1-gpio 
sudo modprobe w1-therm

Load module on startup: 

sudo nano /boot/config.txt

enter the following at the end: 

dtoverlay=w1-gpio
gpiopin=4

The measured temperature can be read from the file: 

/sys/bus/w1/devices/28-0217b20f16ff/w1_slave
